📋 Términos de Uso

Duvi Stream OS es una herramienta de productividad personal para organizar y reproducir videos. El uso de esta aplicación implica la aceptación de los presentes términos.

🛡️ Política de Privacidad

Esta aplicación no recopila ni transmite datos personales. Toda la información (biblioteca, configuración, API keys) se almacena exclusivamente en el localStorage de su navegador. Las API keys se usan únicamente para comunicarse con el proveedor IA elegido por el usuario.

🎬 Contenido de Terceros

Duvi Stream OS no aloja contenido multimedia propio. Actúa como organizador de URLs externas. El usuario es el único responsable de las URLs que añade y del cumplimiento de los derechos de autor de dicho contenido.

🤖 Inteligencia Artificial

El asistente de IA (Media Concierge) usa APIs de terceros (Groq, Gemini). El desarrollador no controla ni es responsable del contenido generado por estos modelos. El usuario debe verificar la exactitud de las respuestas.

⚠️ Limitación de Responsabilidad

La aplicación se proporciona "tal cual" sin garantías. No se garantiza disponibilidad, precisión o adecuación para ningún propósito específico.

☕ Donaciones

Las donaciones son voluntarias y no garantizan servicios adicionales. Se usan para financiar el desarrollo continuado.
